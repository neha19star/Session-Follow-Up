Word Scramble Game (in C)

A vocabulary-based terminal game made using C. Users are prompted to unscramble letters to form a valid word. A fun and educational mini-project!

  • Random word logic & scramble function
  • User input comparison with solution
  • Displays result and feedback
#include <stdio.h>
#include <string.h>

int main() {
  char word[] = "agenda";
  char guess[20];

  printf("Unscramble the word: rngdea\n");
  scanf("%s", guess);

  if (strcmp(word, guess) == 0)
    printf("Correct!\n");
  else
    printf("Try again!\n");

  return 0;
}
